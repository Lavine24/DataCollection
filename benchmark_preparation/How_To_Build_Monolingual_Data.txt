How to collect monolingual data and build the monolingual repository

X) Script for downloading moonolingual data from data.statmt.org (UEdin), for all languages but English
  bash scripts/get_monolingual_corpus.sh [de|es|fr|it]

X) Script for downloading moonolingual data from data.statmt.org (UEdin), for English only
  bash scripts/get_monolingual_corpus.sh en

X) Script for reducing the size of the moonolingual data, for all languages but English
  bash reduce_monolingual_corpus.sh [de|es|fr|it] 

X) Script for reducing the size of the moonolingual data, for English only
  bash reduce_monolingual_corpus_English.sh en
